Finder hides files and folders that start with a dot.

The actual project files are already included here:
- ../.gitignore
- ../.streamlit/config.toml
- ../.streamlit/secrets.toml.example

To see them in Finder, press Command + Shift + Period.

Do not create or commit .streamlit/secrets.toml with a real API key. For Streamlit Cloud, add OPENAI_API_KEY in the app's Secrets settings instead.
